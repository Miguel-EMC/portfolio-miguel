## Acerca del Proyecto

Sistema empresarial desarrollado con Django y React Native para facturación electrónica, gestión de clientes e integración completa con el SRI (Servicio de Rentas Internas) y otros organismos gubernamentales del Ecuador. La plataforma gestiona el ciclo completo de facturación desde la creación hasta el envío al Estado y el archivado.

## Desafíos

- Construir un sistema de procesamiento de pagos seguro y conforme con la normativa del SRI del Ecuador
- Crear aplicaciones móviles multiplataforma que funcionen de manera confiable en Android e iOS
- Implementar seguimiento de transacciones en tiempo real y actualizaciones de estado
- Gestionar envíos masivos concurrentes de facturas a las APIs gubernamentales

## Soluciones

- Integración con pasarelas de pago compatibles con PCI y la API de facturación electrónica del SRI
- Uso de React Native para una única base de código orientada a iOS y Android
- Implementación de conexiones WebSocket para actualizaciones de estado de transacciones en tiempo real
- Construcción de un mecanismo de reintento basado en colas para envíos fallidos a las APIs gubernamentales

## Resultados

- Procesamiento de más de $2M USD en transacciones facturadas
- Obtención de la certificación de cumplimiento PCI DSS completa
- Reducción de errores en el procesamiento de facturas en un 95% respecto al proceso manual anterior
- Integración con 3 sistemas gubernamentales diferentes (SRI, IESS, Ministerio de Trabajo)
